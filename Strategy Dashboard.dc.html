TIER 1 Primary
Erasmus Mundus Joint Master's — EU
AI / machine learning / data science tracks · promoted above China
✓ UoP BSc is the qualifying credential IELTS / TOEFL GRE + portfolio only if asked
Key advantage: the online UoPeople CS degree is recognised in its awarding country (US-accredited), so here it is the actual qualifying credential — the opposite of the China case, where it gets demoted to a mere attachment.
Eligibility: bachelor's recognised in awarding country — UoP likely qualifies, but confirm per program by emailing coordinators.
Prep: IELTS or TOEFL almost always required; GRE and portfolio only if a specific program asks, not by default.
Timeline: 2027 practice attempt → 2028 serious attempt, GPA improvement in progress.

TIER 2 Backup — heavily constrained
Master's in China
not a clean option — two independent blockers must both clear
Blocker 1 · fit Blocker 2 · CSC 3-year gap
Blocker 1 — fit: even with the Dagon BA English as the recognised degree plus UoP CS and a portfolio as evidence, admission is only realistic at accepting-tier universities (mid-tier, English-taught, cross-field-friendly) — closed at top technical programs.
Blocker 2 — funding: CSC has a 3-year gap rule. If the language year is CSC-funded, a CSC-funded master's is blocked for 3 years. A funded China master's stays possible only if the language year is funded by CIS instead of CSC.
